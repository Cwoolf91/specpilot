### Recording tips

- Use a built-in VS Code theme (**Dark+** or **Light+**) so screenshots feel native
- Capture just the panel: `Cmd+Shift+4` then `Space` on macOS grabs a single window
- GIFs: [Kap](https://getkap.co/) exports compact GIFs; keep under 10 seconds and 8 MB
- Anonymize real Jira keys/emails before publishing
