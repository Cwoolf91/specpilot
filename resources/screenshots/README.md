## Tips for recording

- VS Code theme: use **Dark+** or a common built-in theme so screenshots feel native
- Window chrome: capture just the panel, not the whole desktop — `Cmd+Shift+4` then `Space` on macOS to grab a single window
- GIFs: tools like [Kap](https://getkap.co/) (mac) or [ScreenToGif](https://www.screentogif.com/) (win) export compact GIFs. Keep under 10 seconds and 8 MB.
- Anonymize: blur or swap out real Jira keys/emails before publishing
